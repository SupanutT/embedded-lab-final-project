## Getting Started
### Prerequisites
  ```sh
  npm install
  ```
### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/SupanutT/embedded-lab-final-project.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter your Firebase Config in `/src/script/firebase.js`
   ```js
   const firebaseConfig = {<ENTER YOUR Config>};
   ```
